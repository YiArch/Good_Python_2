Подвиг 6. В программе объявлена функция для вычисления частного двух чисел:

def get_div(x, y):
    try:
        res = x / y
        return res
    except ZeroDivisionError:
        res = 100
        return res
    finally:
        res = -1
        print(f"finally: {res}")

Выберите все верные утверждения, связанные с этой программой.
